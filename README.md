This Python project demonstrates basic Natural Language Processing (NLP) tasks: tokenization, sentiment analysis, and named entity recognition (NER).

## Features

1. **Tokenization** – Splits text into individual words.
2. **Sentiment Analysis** – Determines polarity (positive/negative) and subjectivity (fact/opinion) using TextBlob.
3. **Named Entity Recognition (NER)** – Identifies entities like persons, locations, organizations using spaCy.
